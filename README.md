1–2 Type D units per patch form a distributed IR array covering fingertips, palm, and joints. Binary patterns indicate obstacle location:
- Single node: obstacle directly ahead

### 4.2 Three-Stage Pseudo-Analog Sensing
To retain binary simplicity while achieving graded force feedback, TacNode uses a **discrete threshold array** that converts continuous force into 3‑bit logic:

Three Type A units per cluster:
- **L (Low)**: 0.1 N — touch warning, deceleration
- **M (Mid)**: 1 N — stable contact, grasp lock
- **H (High)**: 10 N — overload protection, retract

Controller uses lookup-table logic:
- `[0,0,0]`: Safe, full speed
- `[1,0,0]`: Approaching, slow down
- `[1,1,0]`: Perfect contact, lock motor
- `[1,1,1]`: Overload, retract 2 mm

### 4.3 Cross Vector Time-Series Sensing
Each TacNode uses a **cross layout (N‑S‑W‑E)**. By measuring time differences ΔT between quadrants, the system infers surface normal, curvature, and contact direction.

Example: object slides from left → W triggers first, then E after Δt.
This enables automatic alignment and adaptive manipulation without 3D modeling.

### 4.4 CPU-Free Hardware Direct-Drive Closed Loop
TacNode’s most revolutionary innovation: it eliminates the traditional sensor‑ADC‑CPU‑driver chain and uses **potential-balanced differential actuation**.

**Principle**:
1. Left (W) and Right (E) connect directly to H-bridge for yaw control
2. North (N) and South (S) connect to pitch control
3. If W=1, E=0: voltage difference drives motor until symmetry
4. When W=1, E=1: voltage difference = 0; motor brakes naturally

This is **pure physics closed-loop control** with electron-speed latency. The robot automatically aligns to any surface without prior shape knowledge.

Instead of friction cone computation, TacNode uses **event-driven geometric locking**:

- When all nodes = `1`, global hardware interrupt triggers
- Interrupt cuts motor power **directly**, bypassing OS

This **geometric closure** eliminates over‑grasping fundamentally, independent of weight or material.

### 4.6 Millisecond Slip Detection
Once locked, any node falling from `1` to `0` indicates micro-slip. The system triggers corrective tightening within **10 ms**, achieving real-time anti-slip stability.

我们设计的管路逻辑连接，正是学术界的**微流控逻辑（Microfluidic Logic）**，用流体本身完成了信号处理与逻辑判断，无需任何电子芯片：

## 3. 核心理念：形态学计算与“纯物理外挂”
TacNode的核心哲学是**形态学计算(Morphological Computation)**——利用物理系统自身的结构与材料特性来完成计算任务，而非依赖通用处理器。我们将其称为“纯物理外挂”架构：传感器不再是单纯的信号采集器，而是具备原生计算能力的智能单元。

在传统触觉感知链路中，信号处理遵循“物理刺激→模拟信号→ADC→降噪算法→CPU→电机驱动”的漫长路径。TacNode彻底颠覆了这一范式：
- **源头滤波**：利用材料的物理阈值特性，在信号产生的瞬间完成杂波过滤与模数转换
- **本地计算**：通过硬件逻辑门与差分电路实现特征提取与闭环控制
- **事件驱动**：仅在物理状态发生变化时产生信号，平时处于零功耗休眠态

这种架构将触觉系统的延迟从“大脑反应级”(百毫秒)降至“脊髓反射级”(微秒级)，同时将算力消耗降低了数个数量级。

## 4. 核心技术架构
### 4.1 硬件级物理滤波器阵列
TacNode内部不包含任何负责降噪的DSP芯片。我们通过在硅胶真皮层中植入四种具有特殊物理阈值的微型感知单元，在源头实现了硬件级多模态滤波。

### 四种硬件级物理滤波器特性总览
| 类型 | 核心材料 | 滤波类型 | 感知范围 | 响应延迟 | 静态功耗 | 核心功能 |
|------|---------|---------|---------|---------|---------|---------|
| **Type A** | 量子隧道复合材料(QTC) | 机械高通滤波器 | 0cm（接触） | <1μs | <0.1μW | 接触测力、抓稳锁死、滑动检测 |
| **Type B** | 压电陶瓷薄壳 | 机械带通滤波器 | 1-3cm | <10μs | <0.1μW | 近距气流预警、精确定位 |
| **Type C** | PVDF热释电薄膜 | 温度突变滤波器 | 0-5cm | <1ms | 0μW | 零功耗温度感知、防烫预警 |
| **Type D** | 940nm红外对管 | 光学带通滤波器 | 1-10cm | <50μs | <1μW | 远距防撞预警、一级减速 |

#### 4.1.1 Type A：基于QTC材料的机械高通滤波器
**物理机制**：量子隧道复合材料(QTC)在自然状态下处于绝对绝缘态。只有当其受到的物理挤压突破特定应力阈值时，材料内部导电颗粒间距缩小至量子隧穿尺度，电阻瞬间呈指数级暴降，转变为良导体。

**滤波效果**：天然免疫环境振动、轻微摩擦和电机高频噪音。只有当抓取力度达到“有效接触”阈值时，才会输出干净的布尔信号`1`。通过调整QTC材料的掺杂比例与外壳硬度，可精确设定不同的触发阈值。

#### 4.1.2 Type B：基于压电陶瓷薄壳的机械带通滤波器
**物理机制**：采用中空脆薄压电陶瓷壳体结构，具有固定的固有谐振频率。当机械手以特定速度靠近目标时，手掌与物体间的空气被挤压形成特定频率的气流波，只有当气流波频率与壳体谐振频率吻合时，压电陶瓷才会产生剧烈形变并释放电压脉冲。

**滤波效果**：精确过滤杂乱的环境微风，只捕捉“即将碰撞”的特定气流，实现1-3厘米范围内的非接触式超距预警。

#### 4.1.3 Type C：基于热释电材料的温度突变传感器
**物理机制**：钛酸钡热释电晶体无需供电，只要周围温度发生微小变化，表面就会自动产生电荷。

**感知能力**：零功耗感应0.5℃以上的温度突变，可区分冷热物体，同时具备防烫预警功能。

### 4.1.4 Type D：基于调制红外的光学带通滤波器（远距离防撞预警）
Type D是TacNode感知体系的**最外层远距离预警单元**，覆盖1-10cm探测范围，负责在机械手高速运动时提前发现障碍物，触发一级减速，避免高速碰撞。它与Type B（1-3cm气流预警）、Type A（接触测力）形成三级递进的感知防线，实现从"远距预警→近距精确定位→接触锁死"的全流程无缝衔接。

#### 物理机制与硬件级双重滤波
Type D摒弃了传统红外测距依赖的ADC采样与软件计算方案，采用**纯硬件开关型架构**，所有信号处理在传感器内部完成，直接输出标准TTL/CMOS布尔电平。其核心是**光学+电学双重硬件滤波机制**，彻底解决了传统红外传感器易受环境光干扰的痛点：
- **光学带通滤波**：传感器表面集成940nm±20nm窄带滤光片，仅允许与发射端波长一致的红外光通过，阻隔可见光及其他波段的红外辐射，可抵御10万lux日光直射（正午阳光强度）的干扰
- **电学调制解调滤波**：发射端以38kHz固定频率发射脉冲红外光，接收端内部集成专用硬件带通滤波器与AC耦合电路，仅允许38kHz±1kHz的交流调制信号通过，同时阻断环境光产生的直流分量与低频漂移

#### 感知能力与阈值控制
- **探测距离**：1-10cm硬件可调，通过调整发射端限流电阻或比较器参考电压实现，无需软件校准。量产时可通过激光修调电阻统一校准阈值，保证批次一致性
- **响应延迟**：<50μs，远快于传统ToF传感器的毫秒级响应
- **输出特性**：纯二值化输出，物体进入设定距离内输出`1`，否则输出`0`，主控仅需读取IO口电平，无需任何计算
- **光束角**：15°窄光束设计，避免相邻节点串扰，可精确判断障碍物的方位

#### 分级低功耗设计
Type D采用**事件驱动的分级功耗控制**，最大化系统能效：
- **休眠模式**：平时红外发射管以1Hz极低频率发射脉冲，静态功耗<1μW
- **激活模式**：当检测到物体进入探测范围后，自动切换至38kHz高频工作模式，同时通过硬件信号线触发同区域的Type B压电节点进入激活状态
- **休眠唤醒**：物体离开探测范围3秒后，自动切回休眠模式

#### 分布式阵列布局与协同感知
每个TacNode贴片集成1-2个Type D节点，在机械手表面形成**分布式红外探测阵列**，无死角覆盖指尖、掌心、关节等关键部位。多个节点的二值信号组合可粗略判断障碍物的方位和大小：
- 单个节点触发：障碍物位于该节点正前方
- 左侧节点触发：障碍物在机械手左侧
- 多个节点同时触发：前方有大面积障碍物

#### 与其他感知单元的协同工作流
Type D作为整个感知系统的"第一道防线"，与其他单元紧密配合，形成完整的抓取闭环：

单TacNode节点采用**十字形(N-S-W-E)空间布局**或三角形，将单点感知升维为表面法向矢量感知。当机械手接触物体时，通过四个方位的触发时间差ΔT，系统可瞬间判断物体表面的切线方向与曲率。

**工作原理**：
1. 将十字阵列的左(W)与右(E)传感器作为正负极，直接接入控制指尖偏航角的电机H桥电路
2. 将上(N)与下(S)传感器接入控制俯仰角的电机
3. 当左侧触碰物体(W=1，E=0)时，W与E之间产生电压差，直接驱动电机向右偏转
4. 当手指偏转至与物体完美贴合时(W=1，E=1)，两侧电势相等，电压差瞬间归零，电机自动刹车锁死

3. 中断信号无需经过上位机操作系统，直接切断关节电机动力或启动机械制动器

这种“几何锁死”而非“力矩挤压”的抓握方式，从物理层面上杜绝了过度抓取导致的物体碎裂，且不受物体重量与材质的影响。
